#Web Market

** Functionality: **

+ registration
+ authentication
+ display profile
+ cart profile
+ concat cart (user, not authenticated)
+ change cart (increment, decrement, remote)
+ create order

** Tools: **
JDK 11, Spring 5, JPA 2 / Hibernate 5, AngularJS, Lombok 1, JUnit 4,  Maven 3, Git, Tomcat 9, PostgreSQL /H2 1.4, IntelliJIDEA.

** Notes: **
SQL init is located in the db/migration/V1__init.sql
